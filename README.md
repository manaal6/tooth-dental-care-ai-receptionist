## Tooth Dental Care — AI Reception Agent for tooth Dental Care (Waterloo)

This is a real-time AI voice reception agent demo, customized for **tooth dental care** (toothlondon.co.uk), built to handle inbound calls: greeting callers, understanding what treatment/appointment they need, collecting name & phone number, confirming details, and logging the lead for the front-desk team to follow up — 24/7, even outside the 8am–8pm hours.
